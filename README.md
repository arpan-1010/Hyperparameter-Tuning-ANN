🧠 ANN Hyperparameter Tuning using Optuna

This project implements an Artificial Neural Network (ANN) using PyTorch and performs hyperparameter tuning with Optuna on the Fashion MNIST dataset.

📌 Features
Custom Neural Network built from scratch using PyTorch
Hyperparameter tuning using Optuna
Supports multiple optimizers (Adam, SGD, RMSprop)
Includes Batch Normalization and Dropout
GPU support (if available)
Visualization of sample dataset images
📂 Dataset
Dataset used: Fashion MNIST
